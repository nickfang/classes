# Finally Scripting

- Basic scripts

`hello` - say hello

```bash
echo hello
```

`greet` - say hello to a user

``` bash
name='dave'
echo "hello $name"
```

`num-items`

``` bash
name='dave'
items=5

echo "user $name has $items items"
```

`loop` - Loop over a set of items

``` bash
for thing in foo bar baz bat; do
    echo "thing is $thing"
done
```
